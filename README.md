## Overview

This project involved redesigning a Sri Lankan travel and tourism web application while preserving its original business concept and services.

The focus was on improving:

- User Experience (UX)
- Mobile Responsiveness
- Navigation Structure
- Content Presentation
- AI-powered Customer Assistance

## Technologies & Tools

- Lovable.dev
- XetelAI
- Prompt Engineering
- Responsive Web Design
- AI Chatbot Integration

## Key Features

- Modern tourism website design
- Responsive layout for desktop and mobile devices
- AI-powered travel assistance chatbot
- Improved content organization
- Enhanced customer engagement

## Project Outcome

The redesigned platform provides a more intuitive user experience, improved accessibility across devices, and enhanced customer interaction through AI-powered assistance.
